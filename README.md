# 🎵 MoodPrint — Music Mood Fingerprinting via LDA

**Dimensionality reduction meets psychoacoustics.**  
180 acoustic features → LDA → 3 discriminant axes → 4 mood clusters.

Grounded in **Russell's Circumplex Model** of affect (valence × arousal).  
Dataset: **DEAM** (2058 songs, ~100 academic citations).

---

## Architecture

```
Audio (.mp3)
    ↓
librosa feature extraction   180 features / song
    ↓
StandardScaler               zero mean, unit variance
    ↓
LinearDiscriminantAnalysis   180D → 3D (C−1 axes)
    ↓
Fingerprint Space            interactive 3D scatter
    ↓
Mahalanobis classification   Happy / Energetic / Calm / Sad
```

## Feature Set (180 total)

| Group             | Dim | Description                         |
|-------------------|-----|-------------------------------------|
| MFCCs (mean+std)  |  80 | Timbre, texture                     |
| Chroma (mean+std) |  24 | Harmonic / note content             |
| Spectral Contrast |  14 | Peak vs valley energy per band      |
| Tonnetz           |  12 | Harmonic relations (tonal centroid) |
| Mel Bands         |  24 | Mel spectrogram (12 compressed)     |
| Spectral Shape    |  10 | Centroid, BW, rolloff, ZCR, RMS     |
| Rhythm            |   4 | Tempo, pulse clarity, onset rate    |
| Dynamics          |   4 | RMS var, dynamic range, crest       |
| Harm/Perc Ratio   |   4 | Harmonic vs percussive energy       |
| Pitch Stats       |   4 | Mean, std, median, range            |

## Mood Classes (Russell's Circumplex)

```
High Valence + High Arousal  →  Happy      (bright, uplifting)
Low  Valence + High Arousal  →  Energetic  (tense, intense)
High Valence + Low  Arousal  →  Calm       (peaceful, soothing)
Low  Valence + Low  Arousal  →  Sad        (melancholic, subdued)
```

## Quick Start

### 1. Get the Dataset

Download DEAM from Kaggle:  
https://www.kaggle.com/datasets/imsparsh/deam-mediaeval-dataset-emotional-analysis-in-music

Extract so the structure looks like:
```
deam/
    MEMD_audio/      ← .mp3 files
    annotations/
        static_annotations_averaged_songs_1_2000.csv
```

### 2. Sort audio into mood folders

```bash
pip install pandas numpy
python prepare_deam.py --deam_dir deam/ --output_dir data/
```

### 3. Train the LDA model

```bash
pip install librosa scikit-learn joblib
python train.py --data_dir data/ --model_dir models/
```

Training takes ~30–90 min depending on your machine (2058 files × 30s each).  
RTX 3050 laptop: ~35 min.

### 4. Start the backend

```bash
pip install -r backend/requirements.txt
uvicorn backend.main:app --reload
# → http://localhost:8000
```

### 5. Start the frontend

```bash
cd frontend
npm install
npm run dev
# → http://localhost:5173
```
